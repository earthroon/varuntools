# VT-UI-22R6

Custom Minimal Video Controls / Native Chrome Replacement / Surface Play Pause Progress Bar / No Browser Controls UI

## Goal

- Do not render browser native controls by default.
- Add a minimal custom control bar inside the video clip.
- Keep tap/click surface play/pause.
- Keep Enter and Space keyboard play/pause.
- Keep explicit native controls opt-in when controls="true" is provided.
- Keep R3 inner 1px inset clip and 1.5px radius.
- Keep R4 mobile portrait width guard.
- Keep loadedmetadata fallback.
